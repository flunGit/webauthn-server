

### 发布流程:
	验证发布:
		访问：https://www.npmjs.com/package/@flun/webauthn-server

---

### 维护:
	1. 更新版本
```bash
		#打包扩展
			npm pack

		# 测试安装(带详细输出)
			npm i @flun/webauthn-server-1.0.1.tgz --foreground-scripts

		# 小版本更新（bug修复）
			npm version patch
			npm publish

		# 功能更新
			npm version minor
			npm publish

		# 重大变更
			npm version major
			npm publish
		# 或者2条命令同时执行(简写:以小版本更新为例)
			npm version patch && npm publish

		.\publish.ps1                       # 发布脚本


```
	2. 撤回发布（72小时内有效）
```bash
		npm unpublish @flun/webauthn-server@1.1.10  # 删除特定版本
		npm deprecate @flun/webauthn-server "该包已迁移至 ... 下,请改用：npm install 新包名" # 弃用包
		npm unpublish @flun/webauthn-server --force # 删除整个包
		npm view 旧包名 # 弃用验证
```
